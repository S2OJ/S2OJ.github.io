## 头像

S2OJ 支持 Gravatar 和 QQ 两种头像来源。默认情况下会使用 Gravatar 头像，但可以在「更改个人信息」页面选择使用 QQ 头像。

使用 QQ 头像前必须先填写 QQ 号。

## 更改个人信息

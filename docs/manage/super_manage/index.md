# 首页管理

## 公告

![](https://arina.loli.net/2022/10/23/wuMxpUaRBEYQzPF.png)

在「首页管理」页面的「公告」选项卡下，可以浏览并编辑、移除所有公告。

![](https://arina.loli.net/2022/10/23/qsgvBi2rp9ze1Tt.png)

在页面底部有一个名为「添加/修改公告」的表单，可以新增公告或者修改现有公告的置顶级别。

## 倒计时

在「倒计时」选项卡下可以对倒计时进行管理。

![](https://arina.loli.net/2022/10/23/zmxM2gLI4PfdQK8.png)
